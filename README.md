Project 1 — Twitter Handles Analyzer
Description

A Ruby command-line program that analyzes a large dataset of Twitter handles.
It provides multiple statistics and transformations through an interactive menu.

Features

Count the total number of handles

Find the shortest handle

Count handles with exactly 5 characters (excluding @)

Count handles starting with an uppercase letter

Sort handles alphabetically

Sort handles by length

Find the position of a specific handle (@epenser)

Group handles by length

Technologies

Ruby

Arrays, Enumerables (map, select, sort_by, tally)

CLI menu interaction

Project 2 — Cryptocurrency Dataset Processor
Description

A Ruby script manipulating a very large dataset of cryptocurrencies and their prices.
The project focuses on data handling, indexing, and parallel arrays (crypto and prix).

Features

Store and manipulate hundreds of cryptocurrency names

Associate cryptocurrencies with their prices

Practice array traversal and data alignment

Learn to manage large datasets in Ruby

Technologies

Ruby

Arrays

Data parsing and manipulation

How to Run
ruby your_file_name.rb
